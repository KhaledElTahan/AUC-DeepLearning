Problem Statement

The classical game Snake was a world’s amusement since 1977. The game is about challenging a snake on the screen to be able to manage moving around a 2 dimensional grid to get rewards (apples/food). The agent should follow the following rules:

    1. Upon eating food, the snake gets its body grown
    2. The snake has the choices of either:
      A. Continuing to move forward in the given direction.
      B. Take a left turn.
      C. Take a right turn.
    3. The snake dies upon contact with its own body or a wall.
Deep Learning Challenge:
    The challenge is essentially creating using deep learning to build a neural network that would learn playing the game maximizing the score in a form of intersection between modern Deep Learning and Artificial Intelligence.

Simulation

A simulation will be used to simultae the process of the game and train the model. It will generate a random position for the apple each time it eaten and for each game. It will take direction of next movement of snake from neural network and send back a feedback about the status of snake.

Proposed Updates

This study takes an approach which merges both neural networks and a genetic algorithm to get more accuracy.  The genetic algorithm (GA) and neural network(NN) play the snake game, and Instead of backpropagation, Weights will be updated using GA to find the best parameters. The Implementation will go through these steps:

  • Determine neural network architecture.
  • Creating an initial population and fitness function (can use some factors from similar models which attack the same problem)
  • Play a game for each individual in the population and sort each individual in the population based on the fitness function score.
  • Select a few top individuals from the population and create the remaining population from these top selected individuals using Crossover and mutation.
  • The new population is created (meaning the next generation).
  • And so on..

    • The fitness function will be computed based on some rules:
  • The number of points snake collected (+ve reward)
  • Snake hits itself or the border (-ve reward)
  • Snake is just moving around itself (-ve reward)

    • Input of neutal Network:
  • Is the right direction blocked
  • Is the left direction blocked
  • Is the front direction blocked 
  • Snake length
  • The direction of the head of the snake
  • The Vector direction between head of snake and apple

    • Population Size: 300 Number of Generations: 500

    Conclusion

    This project is a proof of concept which is introduced in December 2017 by Uber AI Labs. It published a paper which is related to the topic of neuroevolution, a practice where deep neural networks are optimized by evolutionary algorithms. One of them is “Deep Neuroevolution: Genetic Algorithms are a Competitive Alternative for Training Deep Neural Networks for Reinforcement Learning” which deduces that: “Using a simple Genetic Algorithm (GA) it is possible to optimize DNNs and that GAs are a competitive alternative to gradient based methods when applied to Reinforcement Learning (RL) tasks such as learning to play Atari games, simulated humanoid locomotion or deceptive maze problems.”
